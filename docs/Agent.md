# AGENTS.md

This file provides context about this Obsidian vault for AI agents.

## Vault Overview

この保管庫は、AI（OpenAI, Luma, ElevenLabs等）とRemotionを統合した次世代の動画生成自動化インフラ「Remotion Orchestra」の開発と管理を目的としています。

単なる動画編集プロジェクトではなく、世界中の誰もが高度な動画制作を完結できる「動画編集の自立化」を実現するための汎用的な動画生成エンジンの構築を目指しており、その設計思想、アーキテクチャ、開発ログが記録されています。

## Organization

プロジェクトの構造は機能ごとに明確に分離されたモジュール形式を採用しており、コードベースと密接に連動しています。

- **src/**: 再利用可能なコンポーネント、タイムライン定義、およびマルチモーダルAIを統合する「glue（接着剤）」コードを格納。
- **skills/**: AIエージェントがユーザーの意図を汲み取るためのカスタムスキル定義。
- **インフラ管理**: Cloudflare R2を用いたグローバル配信を前提としたアセット管理。
- **日次管理**: `2026-01-30` のような日付ベースのノートによる進捗管理と、AIエージェント（Jules）への指示書（`.antigravity`）による開発支援の自動化。

ノート間の接続は、技術スタックの統治規約とプロジェクトのビジョンを中心に構造化されています。

## Key Topics

- Remotion Orchestra (動画生成自動化インフラ)
- Independence of Video Editing (動画編集の自立化)
- AI Orchestration (Luma, OpenAI, ElevenLabsの統合)
- Cloudflare R2 (グローバル・アセット・配信戦略)
- Google Style Guides / Biome / Ruff (厳格なコーディング規約)
- Jules (開発パートナーとしてのAIエージェント)
- Universal Glue Code (汎用プラグイン設計)

## User Preferences

ユーザー（sumash）は、極めて高い技術的規律と明確なビジョンを持っています。

- **厳格な規約**: Google Style Guideへの準拠、型安全性の確保（`any`の排除）など、プロフェッショナルなエンジニアリング品質を求めています。
- **汎用性の重視**: 特定の案件に特化するのではなく、常に「全ユーザーのための汎用エンジン」としての設計を優先します。
- **パートナーシップ**: AIを単なるツールではなく、創造性を拡張する「Jules」というパートナーとして定義し、自律的な支援を期待しています。
